# Annotation

Spring 提供了大量的**注解**`@Annotation`用于简化配置及使用`Java`替代`xml`。

如：

* 用`@Component`及其扩展`@Service`、`@Controller`等替代`<bean>`，标记一个类为组件类;
* 用于替代`<context:component-scan>`配置完成组件扫描的`@ComponentScan`;
* ……
